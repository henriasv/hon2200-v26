# Endringslogg for HON2200

Denne mappen dokumenterer endringer i kurset HON2200 i tråd med SOTL-prinsippene (Scholarship of Teaching and Learning).

## Formål

Systematisk dokumentasjon av:
- Hva som endres i kurset
- Hvorfor endringen gjøres (pedagogisk begrunnelse)
- Evidens som ligger til grunn (evaluering, forskning, erfaring)
- Forventet effekt
- Hvordan effekten kan måles

## Struktur

```
endringslogg/
├── README.md                    # Denne filen
├── v26_planlegging.md           # Planlegging for vår 2026
├── evalueringer/
│   └── evaluering_v25.md        # Emneevaluering vår 2025
└── endringer/
    └── YYYY-MM-DD_beskrivelse.md  # Individuelle endringsdokumenter
```

## Mal for endringsdokumenter

Hver endring dokumenteres med følgende struktur:

```markdown
# [Tittel på endring]

**Dato:** YYYY-MM-DD
**Ansvarlig:** [Navn]

## Hva ble endret
[Beskrivelse av endringen]

## Hvorfor (pedagogisk begrunnelse)
[Begrunnelse for endringen]

## Evidens
[Evaluering, forskning, eller erfaring som støtter endringen]

## Forventet effekt
[Hva vi forventer skal bli bedre]

## Hvordan måle effekt
[Hvordan vi kan vurdere om endringen fungerte]

## Oppfølging
[Dato og resultat av oppfølging]
```
