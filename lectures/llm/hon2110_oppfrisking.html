Penn-og-papir-oppgaver for å friske opp grunnleggende programmering og statistikk

Author

Henrik Sveinsson med Claude Opus

Last ned PDF-versjon

Oppgave 1: Les og forstå kode

a) Hva skriver koden ut?

For hver kodeblokk, skriv hva som printes til skjermen.

x = 10
y = "5"
z = x + int(y)
print(z)
navn = "Ada"
alder = 30
print(f"Jeg heter {navn} og er {alder} år gammel")
tall = [10, 20, 30, 40, 50]
print(tall[0])
print(tall[-1])
print(tall[2])
for i in range(3):
    print(i * 2)
def doble(x):
    return x * 2

resultat = doble(5)
print(resultat)

b) Finn feilen

Hver kodeblokk inneholder én feil. Beskriv feilen og skriv den rettede koden.

x = "5"
y = 10
z = x + y
print(z)
liste = [1, 2, 3, 4, 5]
for i in range(len(liste) + 1):
    print(liste[i])
def hils():
    print("Hei verden")

print(hilsen())
tall = [1, 2, 3, 4, 5]
gjennomsnitt = sum(tall) / len(tall)
print("Gjennomsnitt:", gjennomsnittet)

Oppgave 2: Skriv kode

a) Variabler og lister

  1. Lag en liste ukedager som inneholder alle 7 ukedager på norsk (starter med “mandag”).

  2. Gitt listen fra a), skriv kode som:

  • Henter ut “onsdag” fra listen
  • Legger til “helg” på slutten av listen
  • Finner antall elementer i listen

b) Betingelser (if-else)

  1. Skriv kode som sjekker om variabelen karakter er bestått (4 eller høyere på skala 1-6) og printer passende melding.

  2. Skriv kode som klassifiserer variabelen temp (temperatur i celsius):

  • Under 0: “Frost”
  • 0-15: “Kjølig”
  • 16-25: “Behagelig”
  • Over 25: “Varmt”

c) Løkker

  1. Skriv en for-løkke som printer tallene 1 til og med 5.

Hint: range(start, slutt) gir tall fra start til slutt-1

  1. Gitt listen navn = ["Ada", "Bob", "Cleo"], skriv en løkke som printer “Hei, Ada!”, “Hei, Bob!”, osv.

d) Funksjoner

  1. Skriv en funksjon kvadrat(n) som returnerer n².

  2. Skriv en funksjon er_partall(tall) som returnerer True hvis tallet er partall, ellers False.

  3. Skriv en funksjon finn_maksimum(liste) som tar inn en liste med tall og returnerer det største tallet.

Hint: Du kan bruke en løkke, eller tenke på om Python har en innebygd funksjon for dette.

  1. Hva er forskjellen mellom print() og return i en funksjon?

  2. Hva vil følgende kode skrive ut?

def legg_til_ti(x):
    return x + 10

resultat = legg_til_ti(5)
print(resultat)
print(legg_til_ti(resultat))

Oppgave 3: Statistikk med kode

a) Gjennomsnitt og median

Du har datasettet: [10, 20, 30, 40, 50]

  1. Skriv kode som regner ut gjennomsnittet av datasettet.

  2. Skriv kode som finner medianen.

  3. Utvid koden din til å håndtere datasettet [10, 20, 30, 40, 50, 200]. Hva skjer med gjennomsnittet og medianen når vi legger til en ekstremverdi?

b) Standardavvik

Du har datasettet: [2, 4, 6, 7, 10]

  1. Skriv kode som regner ut gjennomsnittet.

  2. Skriv kode som regner ut variansen.

  3. Skriv kode som regner ut standardavviket (√varians).

Du har følgende data:

studenter = [

    {"navn": "David", "alder": 21, "karakter": 3.5},
    {"navn": "Elin", "alder": 24, "karakter": 4.8}
]
  1. Skriv kode som finner alle studenter med karakter over 4.0.
